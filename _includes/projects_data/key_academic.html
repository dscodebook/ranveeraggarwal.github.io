Key Academic Projects

Image Quilting for Texture Synthesis and Transfer

MATLAB · Prof. Suyash Awate & Prof. Ajit Rajwade

  • Implemented a pioneering algorithm by Efros and Freeman for generating textures by stitching together small image patches
  • Enhanced the algorithm to render an image with a texture taken from a different image

Lightcuts

C++ | PBRT · Prof. Parag Chaudhuri

  • Implemented a scalable framework for computing realistic illumination based on the research paper ‘Lightcuts’ [SIGGRAPH 2005]
  • Significantly lowered the rendering time for several (>1000) uniformly distributed point light sources

Rendering with Photorealistic Renderman

Renderman · Prof. Parag Chaudhuri

  • Wrote shaders and rendered raytraced scenes using Pixar’s rendering software PRMan
  • The resultant scene elements produced effects like color bleeding, caustics, area lights and soft shadows

Incremental Development of a Compiler

C++ | FlexC++ | BisonC++ · Prof. Amitabha Sanyal

  • Implemented a compiler for a subset of the C language using tools like FlexC++ and BisonC++
  • Sethi-Ullman Algorithm was used to create a syntax tree and optimization was done for control statements

Transformer Rendering and Animation

C++ | OpenGL · Prof. Parag Chaudhuri

  • Modeled, textured and animated (via forward kinematics) a transformer robot from scratch with OpenGL
  • Developed an interactive environment for the keyboard controlled bot with inter-object collisions
  • Used motion captured data in the form of BVH inputs to animate the transformer (via reverse kinematics)

Movie Social Network

Java | HTML | Bootstrap · Prof. N.L. Sarda

  • Developed a social network for movie enthusiasts similar to Goodreads using JSP, Java and PostgreSQL
  • Optimised database schema and minimised redundancy after normalisation and ER modelling

Threading and Scheduling Implementation in GeekOS

C | C++ · Prof. D.M. Dhamdhare

  • Modified the existing scheduler of an experimental OS by adding Multi Level and Round-Robin scheduling
  • Added support for kernel and user level threads and methods for deadlock handling

2D Simulation of an Orrery

C++ | Box2D · Prof. Parag Chaudhuri

  • Simulated a mechanical model of Solar System using gears instead of gravity
  • Used Box2D, an open source physics engine for interaction between mechanical components

Chess with Artificial Intelligence

PLT-Scheme · Prof. Amitabha Sanyal

  • Developed a chess game in PLT Scheme using in-built GUI Toolkit in DrRacket
  • Implemented the Minimax Algorithm with Alpha-Beta Pruning for the AI with a tree depth of 3
